## 📊 Food Delivery Analytics Dashboard (Power BI)

🚀 End-to-End Data Analytics Project | From Raw Data to Business Insights

This project demonstrates a complete data analytics workflow using Power BI — starting from messy raw data to building an interactive, insight-driven dashboard for decision-making.

## Project Overview

This dashboard provides deep insights into :

📦 Order trends
💰 Revenue performance
👥 Customer behavior
🍽️ Restaurant analytics
🚚 Delivery insights

The goal is to transform raw data into actionable business insights. 

## Key Workflow Stages

1️⃣ Data Preparation (ETL & Data Modeling)
- Imported raw Excel data into Power BI
- Cleaned and transformed data using Power Query
- Handled missing values and inconsistencies
## Built a Star Schema Model :
⭐ Fact Table (Orders, Revenue)
📘 Dimension Tables (Customers, Restaurants, Date)

👉 Result : Optimized data model for fast and efficient analysis.


## 2️⃣ DAX Calculations (Business Metrics)

Created key performance measures using DAX :

📦 Total Orders
💰 Total Revenue
🍽️ Revenue per Restaurant
⭐ Average Rating
👍 Average Votes

👉 Focus : Converting raw data into meaningful KPIs.
